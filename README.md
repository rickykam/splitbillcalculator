📱 Split Bill Calculator – User Guide
Welcome to the Split Bill Calculator! This is a lightweight bill-splitting tool designed for travel, group dining, and shared purchases. No login or installation is required—just open the webpage and start using it. All data is stored directly on your phone.

🚀 Getting Started (Install as App)
Although this is a web app, it is recommended to add it to your home screen for the best full-screen experience:

iOS (Safari): Tap the Share button (square with arrow) at the bottom → Select “Add to Home Screen”.
Android (Chrome): Tap the Menu button (three dots) in the top right → Select “Add to Home Screen” or “Install App”.


📖 How to Use
1️⃣ Create a Group (Activity)
When you open the app, you will first see the list of activities.

Tap the “New Activity” button
Enter the activity name (e.g., “Tokyo 5D4N Trip”, “Friday Dinner”) and date
After creating, tap the activity to enter the expense tracking page
Tip: Tap the pencil icon to edit the name, or the trash icon to delete the activity


2️⃣ Set Members
Before adding expenses, it’s recommended to set up participants first.

Go to the “Members” tab at the bottom
Add a member: Enter a name (e.g., “John”) and tap add. You can also optionally enter their PayID/bank details
Set transfer info: Tap the edit button next to a member to add or update PayID—this is useful for final settlements


3️⃣ Add Expenses
This is the main feature. Tap the “＋” floating button in the bottom right to start.


Item Name: Enter the expense (required, e.g., “Lunch”)


Upload Receipt (Camera Feature):

Tap “Camera” to take a photo of the receipt
Tap “Gallery” to upload an existing receipt photo



Who Paid?

By default, one person may have paid everything
For shared payments (e.g., A paid $100, B paid $50), enter amounts next to each name
The system will automatically calculate the total



Who Shares? (Weighted Split)

By default, all members are selected
Tap a name to include/exclude them from sharing
Weighting: If someone consumed more (e.g., 2 portions), tap the “＋” next to their name to adjust their share. The system calculates proportionally



Complete:

Tap “Continue Adding”: Save current entry and reset fields for the next one
Tap “Done”: Save and return to the list




4️⃣ View Settlement
After entering all expenses, go to the “Settlement” tab.


Total Expenses: Shows the overall spending for the activity


Settlement Plan (Minimum Transfers):

The system calculates who owes whom and how much, minimising the number of transactions
If the receiver has PayID set, it will be displayed with a “Copy” button for easy transfer



Individual Summary:

Shows each person’s Share, Paid, and Net Balance clearly




⚙️ Other Features

Language Switch: Tap the 🌐 icon on the top right of the homepage to switch between Traditional Chinese / English
View Receipts: In the expense list, tap the thumbnail to view a larger image if a receipt is attached


⚠️ FAQs & Notes


Where is the data stored?

Data is stored in your phone’s browser local storage
✅ Advantage: Private and secure—no cloud upload
❌ Disadvantage: Data will be lost if you clear browser data. Avoid using in incognito mode



Can multiple users sync?

Currently, this is a single-device version
One person can manage expenses and then share a screenshot of the settlement



Camera not working?

Ensure you are using a HTTPS (secure) connection
Allow browser camera permissions
If previously denied, re-enable it in your phone settings
